## ðŸ“Œ Project Overview

This project demonstrates:

1. **Data Generation & Cleaning:**

   * Synthetic sales data with columns: `Month`, `Sales`, `Advertising`, `Employees`.
   * Saved as `sales_data.csv` for further analysis.

2. **Data Analysis:**

   * Descriptive statistics of the dataset.
   * Insights into monthly sales trends and correlations.

3. **Data Visualization:**

   * Line chart for monthly sales trend.
   * Scatter plot for Sales vs Advertising with Employee count as a hue.

4. **Machine Learning Model:**

   * Linear regression to predict `Sales` based on `Advertising` and `Employees`.
   * Model evaluation using RÂ² score and Mean Squared Error (MSE).

   * Predicted vs actual sales saved as `sales_predictions.csv`.
